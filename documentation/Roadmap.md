
# Project Roadmap

This roadmap outlines planned enhancements and future development goals for the Automated Mechanical Desk Lift system. Contributors are encouraged to review, discuss, and help implement these features.

---

## Upcoming Features
- **Upper and lower limit switches:** Add hardware sensors for precise position detection and safety.
- **Current sensing for stall detection:** Implement motor current monitoring to detect stalls and protect hardware.
- **Height presets:** Enable users to save and recall preferred desk heights.
- **EEPROM-based calibration:** Store calibration and preset data in non-volatile memory for persistence.
- **Soft-start and soft-stop PWM control:** Improve motor control for smoother, quieter operation and increased hardware longevity.
